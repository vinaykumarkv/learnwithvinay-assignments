# Python · AI · Robotics — Complete Curriculum

> A structured, end-to-end learning path from Python fundamentals to autonomous robots and generative AI systems.

---

## Who This Is For

This curriculum is designed for learners who want to go **all the way** — from writing their first Python script to building autonomous robots, training deep learning models, designing PCBs, and deploying agentic AI systems. No prior programming experience is required, though curiosity and persistence are essential.

---

## What You Will Build Toward

By the end of this curriculum you will be able to:

- Write production-quality Python and C++ code
- Query and manipulate databases using SQL and Python
- Understand and apply the mathematics behind machine learning and robotics
- Build, train, and deploy ML, DL, and Generative AI models
- Design multi-modal and agentic AI pipelines
- Program robots using ROS2
- Read schematics, design PCBs, and wire electronics
- Deliver complete, end-to-end AI and robotics projects

---

## Curriculum Structure

| # | Module | Focus |
|---|--------|-------|
| 1 | Python Foundations | Core language, OOP, libraries |
| 2 | SQL & Database with Python | MSSQL syntax, then Python-driven queries |
| 3 | Maths for ML & DL | Linear algebra, calculus, probability, statistics |
| 4 | Machine Learning | Classical algorithms, model evaluation |
| 5 | Deep Learning | Neural networks, CNNs, RNNs, Transformers |
| 6 | Generative AI | LLMs, diffusion models, prompt engineering |
| 7 | Multi-Modal AI | Vision-language models, cross-modal systems |
| 8 | Agentic AI | Agents, tool use, memory, orchestration |
| 9 | Robotics Introduction | Concepts, kinematics, sensors, actuators |
| 10 | Maths for Robotics | Transforms, control theory, state estimation |
| 11 | C++ for Robotics | Systems programming, memory, real-time code |
| 12 | ROS2 | Nodes, topics, services, navigation stack |
| 13 | Electronics & PCB | Circuits, schematics, PCB design, soldering |
| 14 | End-to-End Projects | Integrated AI + Robotics capstone builds |

## How to Use This Curriculum

Each module has its own markdown file with:

- **Learning objectives** — what you will know and be able to do
- **Topics** — detailed week-by-week breakdown
- **Practicals** — hands-on exercises and mini-projects
- **Resources** — recommended books, tools, and references
- **Prerequisites** — what you should complete first

### Recommended Path

```
Python → SQL → Maths for ML → ML → DL → GenAI → Multi-Modal → Agentic
                                                      ↓
                             Robotics Intro → Maths for Robotics → C++ → ROS2 → Electronics/PCB
                                                      ↓
                                            End-to-End Projects
```

The AI track and Robotics track can be studied partially in parallel once you have Python and Maths foundations.
